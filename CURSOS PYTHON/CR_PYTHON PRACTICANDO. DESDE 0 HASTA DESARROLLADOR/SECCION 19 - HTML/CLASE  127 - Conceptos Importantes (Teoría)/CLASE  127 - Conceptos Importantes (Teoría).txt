son un fragmento que indica al navegador realizar una función o tarea en específico, ya que HTML es

el lenguaje con el que una página web se habla con el navegador y a partir de esto se crean funciones

o se crean ciertos procesos.

Entonces estas etiquetas determinan la estructura y a su vez la funcionalidad que va a tener la página

web en HTML.

Otro concepto importante es atributo, y es que un atributo es una cualidad que se le puede agregar

a una etiqueta.

La función del atributo es modificar el contenido que tiene dicha etiqueta o agregar una mejor estructura

a una página web.

Un atributo es nada más una característica, una cualidad que tiene una etiqueta para realizar funciones

extra o en la mayoría de los casos, agregar nuevas nuevas características y nuevos estilos a partir

de CSS.

La estructura de un atributo es la que se ve en la imagen.

El nombre de la etiqueta seguido de un atributo y por último, el valor de dicho atributo.

En algunos casos simplemente se pone un atributo sin comillas.
