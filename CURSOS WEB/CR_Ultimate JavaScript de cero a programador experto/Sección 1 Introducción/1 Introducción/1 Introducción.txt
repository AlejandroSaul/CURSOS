Vino a cambiar el mundo de JavaScript, cómo se ejecuta entonces JavaScript?

Como te contaba, JavaScript fue un lenguaje que se escribió para los exploradores web.

Por ende, todos los exploradores web en la actualidad tienen un motor que se encarga de ejecutar JavaScript.

De hecho lo voy a pintar acá con colorcito azul.

A este motor le vamos a decir que se llama JavaScript Engine o motor de JavaScript.

El motor de Google Chrome se llama V8.

Este motor es de código abierto.

Esto quiere decir que cualquier persona en todo el mundo puede acceder a leer el código fuente de este

motor y por presentar esta característica de ser de código libre de código abierto.

Un desarrollador llamado Ryan Dahl logró embeber este motor de V8 dentro de código escrito en C++.

Al hacer esto, permitió que código JavaScript pueda ser ejecutado en prácticamente cualquier lado gracias

al trabajo de este señor Ryan Dahl es que JavaScript puede ser ejecutado en servidores, aplicaciones

de escritorio, aplicaciones móviles, videojuegos o incluso microcontroladores.
